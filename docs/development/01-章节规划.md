7. 将“可见边界优于脑补全貌”继续落实为各专题统一写作纪律
8. 将“workflow 对象化 / 用户可见真相 / 治理即输入边界”落实为跨章节统一判断标准
9. 继续把五层 prompt 合同、统一预算器与 contract-first 阅读法回灌到主线与 API atlas
10. 继续把主入口、navigation 与问题索引做成和正文同步演进的正式层，而不是正文写完后的附带修补
