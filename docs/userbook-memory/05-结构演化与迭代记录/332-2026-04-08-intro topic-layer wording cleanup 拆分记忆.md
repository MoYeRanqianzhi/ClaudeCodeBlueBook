## 为什么这轮值得单独提交

### 判断一：当前问题不在事实内容，而在“层”这个词把并列切口误写成了上下级关系

`00-导读.md` 当前这里其实想说的是：

- `04-专题深潜`：按真实工作目标组织
- `05-控制面深挖`：按高价值控制面组织

这两者是：

- 并列的组织方式

而不是：

- 上一层 / 下一层

但：

- `再分成两层`

会把读者带向一种错误阅读预期：

- 先上一层，再下一层
- 或者 `05` 比 `04` 更深一级

这和当前 userbook 一直在强调的：

- 按问题类型选入口

不一致。

### 判断二：最小修正不是重写两条 bullet，而是先把层级词拿掉

当前问题不是：

- `04` 和 `05` 的解释错了

而是：

- 前导句用错了层级词

所以这轮最小切片不是动下面两条，

而是把：

- `两层`

改成：

- `两种组织方式`

这样：

- 并列关系被说清
- 其它说明可以原样保留
