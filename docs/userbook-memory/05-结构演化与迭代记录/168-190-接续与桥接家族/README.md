## 第一性原理

这组记忆真正想回答的不是：

- “`168-190` 该按编号怎么顺着翻”

而是：

- 哪些记忆负责把 `168-179` 收成子树
- 哪些记忆负责把 `180-190` 收成三条后继线
- 哪些记忆负责提醒“这里稳定的是阅读骨架，不是所有中间节点都已变成公开能力”

所以这个目录的作用只是：

- 把现有 range map、scope guard 与二层导航收成一个家族入口

不是：

- 再制造一张新的 umbrella map
