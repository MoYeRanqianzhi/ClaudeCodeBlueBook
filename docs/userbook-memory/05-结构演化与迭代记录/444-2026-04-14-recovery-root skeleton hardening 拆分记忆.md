## 苏格拉底式自审

### 问：为什么这轮不是继续补 `213`？

答：因为 `213` 确实也还停在旧模板，但 `212` 更上游，也更像 recovery 簇的 root skeleton。先统一它，收益更高。

### 问：为什么这轮不是继续回头补 `137` 或其他 leaf？

答：因为这些 leaf 页当前最多只是局部证据厚度略软，不是结构漂移。当前更明显的缺口，在 `212` 这种更上游结构页还没切到统一收束模板。

### 问：为什么这也算“优化相关目录结构”？

答：因为 `212` 就是正文目录系统里的更上游骨架页。把它从“旧式结构综述”拉回“统一 skeleton 语气”，本质上是在减少目录系统里的跨层串写。
