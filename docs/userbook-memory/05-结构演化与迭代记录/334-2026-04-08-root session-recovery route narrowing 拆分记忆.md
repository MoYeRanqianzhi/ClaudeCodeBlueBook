## 这轮具体怎么拆

### 判断三：把单页直达收成对 `路径 8` 的单点 handoff

这轮只做一件事：

1. 把根 `README` 里的：
   - `04-专题深潜/12-会话发现、历史检索与恢复选择专题.md`

### 判断四：那一条是“当前对象运营”，不是“先找回以前的工作”

但它们的目标不同：

- 一条偏当前时间线上的对象运营

- 连续性入口簇的联动重写

超出当前最小提交边界。
