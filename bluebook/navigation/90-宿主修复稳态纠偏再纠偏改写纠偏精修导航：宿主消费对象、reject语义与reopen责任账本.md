因为这一层真正新增的问题已经不是：

1. 先修什么、后修什么。
2. 哪个工件在说谎。
3. 哪个模板该怎样重写。

而是：

1. 宿主、CI、评审与交接到底该共享哪些正式对象。
2. 哪些 reject / escalation 语义必须跨消费者复用。

4. 哪些内部实现细节仍然不能被升级成公共 ABI。

这些都更接近：

- host-consumable object surface
- hard reject semantics
- long-term reopen liability ledger
