# 安全专题索引

`security/` 当前包含 139 篇安全章节和 1 个 README；配套速查附录位于 [appendix/README.md](<appendix/README.md>)，当前含 122 篇速查文档。

这个目录研究的不是泛泛的“Claude Code 很安全”，而是安全边界如何被做成分层控制面、状态机、恢复协议和工程迁移制度。

## 核心判断

- Claude Code 的安全性不是单点沙箱，也不是单点分类器。
- 它更像一套分层安全控制面，覆盖启动信任、权限模式、分类器仲裁、受管环境、外部能力收口、恢复资格和宿主边界。
- 真正的设计重点不是“尽量禁用能力”，而是“能力只能沿着正确边界流动”。
