# 使用专题

`guides/` 把源码研究翻成可执行的方法、模板、审读清单和构建手册，不重复产品说明文案。

本 README 只保留 `guides/` 的稳定入口与任务分流；跨目录执行链统一由 [../navigation/README.md](../navigation/README.md) 承担。

## 什么时候进来

- 当你想先学会用 Claude Code，而不是先背对象链。
- 当你已经知道三条母线，想把判断压成模板、清单和审读问题。
- 当你准备把 Prompt、治理或结构判断落成宿主实现、迁移和验证前的手册。
- 当你需要在公开镜像条件下，把源码质量判断压成可执行方法而不是目录体感。

- 如果你只先判断“请求装配控制面怎样落成模板”，从 [99-如何用苏格拉底诘问法审读请求装配控制面：authority chain、protocol transcript与continuation object.md](99-如何用苏格拉底诘问法审读请求装配控制面：authority%20chain%E3%80%81protocol%20transcript%E4%B8%8Econtinuation%20object.md) 进入。
