# 工具索引

以下按 `src/tools.ts` 和工具常量整理。

## 稳定基础工具

| 工具 | 用途 |
| --- | --- |
| `Read` | 读文件、图片、PDF、notebook |
| `Edit` | 原地编辑文件 |
| `Write` | 创建或覆盖文件 |
| `Bash` | 执行 shell 命令 |
| `Glob` | 找文件 |
| `Grep` | 搜文件内容 |
| `WebFetch` | 获取 URL 内容 |
| `WebSearch` | 搜当前信息 |
| `NotebookEdit` | 编辑 Jupyter notebook |
| `TodoWrite` | 会话级待办清单 |
| `TaskStop` | 停止后台任务 |
| `TaskOutput` | 查看后台任务输出 |
| `Skill` | 调用技能 |
| `StructuredOutput` | 输出结构化 JSON |

## 代理与治理工具

| 工具 | 用途 |
| --- | --- |
| `Agent` | 启动子代理 |
| `SendMessage` | 给代理发消息 |
| `EnterPlanMode` | 进入计划模式 |
| `ExitPlanMode` | 提交计划并退出计划模式 |
| `EnterWorktree` | 切入隔离 worktree |
| `ExitWorktree` | 退出 worktree |
| `AskUserQuestion` | 多选式请求用户输入 |

## 任务与团队工具

| 工具 | 用途 |
| --- | --- |
| `TaskCreate` | 创建任务 |
| `TaskGet` | 读取任务 |
| `TaskUpdate` | 更新任务 |
| `TaskList` | 列任务 |
| `TeamCreate` | 创建多代理团队 |
| `TeamDelete` | 解散团队 |

## 扩展与外部资源工具

| 工具 | 用途 |
| --- | --- |
| `ListMcpResourcesTool` | 列 MCP 资源 |
| `ReadMcpResourceTool` | 读 MCP 资源 |
| `ToolSearch` | 搜可用工具 |
| `mcp` | MCP 工具的占位/适配层，运行时由 MCP client 具体化 |

## 平台与条件性工具

| 工具 | 条件 |
| --- | --- |
| `LSP` | 需启用 LSP 能力 |
| `PowerShell` | Windows/PowerShell 条件下 |
| `Config` | 主要见于 ant 构建工具面 |
| `REPL` | 内部/特定构建 |
| `Sleep` | `PROACTIVE` / `KAIROS` |
| `CronCreate` / `CronDelete` / `CronList` | `AGENT_TRIGGERS` |
| `RemoteTrigger` | `AGENT_TRIGGERS_REMOTE` |
| `Brief` / `SendUserMessage` | `KAIROS` 系列 |

- `TungstenTool`
